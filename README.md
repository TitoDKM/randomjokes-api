# RandomJokes API (NodeJS)

Simple API to fetch random joke

## Requeriments
- Node.js version >= 10.3.0

## Installation guide
Install dependencies
 - npm install

Set your settings
 - Change .env.example to .env
 - Change default settings to match your installation

Run locally
 - npm run dev
